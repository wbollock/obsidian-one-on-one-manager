# Plugin Documentation Images

This directory contains screenshots and images used in the plugin documentation.

## Expected Images

The following images should be added to this directory:

- `dashboard.png` - Dashboard view showing person cards with meeting stats and action buttons
- `timeline.png` - Timeline view of all meetings with a person in chronological order
- `meeting_notes.png` - Example of a one-on-one meeting note with structured sections
- `template.png` - Template customization interface
- `goals.png` - Goals tracking interface with progress bars and key results
- `profiles.png` - Person profile management modal

## Image Guidelines

When adding screenshots:
- Use PNG format for clarity
- Capture at a reasonable resolution (1280px width is recommended)
- Ensure no sensitive information is visible
- Use the test vault data for consistent screenshots
- Consider using a neutral Obsidian theme (default light or dark)
